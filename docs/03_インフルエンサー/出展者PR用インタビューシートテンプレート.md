   上記の質問項目をGoogleフォーム（無料）で作成し、出展者の申込完了通知メールや、出展説明会（9/25）の案内時に「QRコード」または「リンクURL」として配布します。スマホから写真も含めて一発で回答・送信できるため、回収率が劇的に上がります。
