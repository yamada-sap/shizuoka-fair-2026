# 📐 インフルエンサー折衝・運用設計キックオフ打合せ資料

---

## 📅 打合せの目的
本ミーティングは、静岡市および周辺エリアの子育てファミリー層（25〜45歳）へ向けた効果的なリーチを目的とし、アサイン会社様とタイアップの「アサインプラン」「投稿頻度」「承認フロー」「法令遵守」「二次利用範囲」を合意し、事故のない円滑な運用体制を確立するために実施します。

---

## 📝 協議事項・確認チェックリスト

### 1. 予算と投稿頻度の最適化（コストパフォーマンス折衝）
インフルエンサー予算「**200,000円**」の範囲内で、最大の効果（リーチ・エンゲージメント）を出すため、すでにアサイン済みの以下2名との投稿内容・回数を調整します。

#### 👤 確定アサインインフルエンサー
1. **静岡県コンシェルジュ**（[@shizuoka_info](https://www.instagram.com/shizuoka_info/)）：県内のお出かけ情報を網羅するファミリー層に強いアカウント。
2. **静岡散歩日記**（[@shizuokaosanponikki](https://www.instagram.com/shizuokaosanponikki/)）：地元のグルメやスポットを親しみやすいトーンで発信するお出かけ特化アカウント。

| 項目 | 協議・質問内容 | 決定・確認事項 | 備考 |
| :--- | :--- | :--- | :--- |
| **確定キャストの投稿数** | 予算200,000円の範囲内で、確定した2名の「フィード投稿数」や「ストーリーズ投稿数」を増やすことは可能か？ | **【220,000円で決定】**<br>プロデューサー判断により見積原価220,000円（税込）で受け入れ。アサイン会社より最終見積書を回収する。 | 当初予算より2万円オーバーとなるが、効果優先で容認。 |
| **複数回セット・ボリューム割引** | 「イベント前告知（1回）」＋「本番来場レポート（1回）」のように、単一キャストに複数回投稿してもらうことでの割引（ボリュームディスカウント）交渉。 | | |
| **追加アサインの可否** | 上記2名に加え、同じ総予算の範囲内で、サブのマイクロ・ナノインフルエンサーを1〜2名追加することは可能か？ | | |
| **実費・諸経費の扱い** | インフルエンサーの交通費、現地食費、体験ブース体験料などは総予算200,000円に含まれるか？ 別途実費精算が必要か？ | **【最終見積に含む形で決定】**<br>受託額FIX案件につき、交通費や食費等の実費は原則として「最終見積220,000円」にすべて含める（別途実費請求は不可）。 | どうしても発生する場合は「雑費」等の予備費枠から捻出するか、上限ルールを事前設定。 |

### 2. 厳格な事前確認 ＆ 承認フロー（手戻り・事故防止）
双方の確認フローおよびバッファを十分に見込んだ進行スケジュールを合意します。

#### 📋 承認ステップと承認連絡ルート
本件は以下の4つの階層で確認が行われるため、連絡バッファを十分に考慮する必要があります。
`[ 1. インフルエンサー ] ➔ [ 2. 協力会社（アサイン元） ] ➔ [ 3. 弊社（SAP） ] ➔ [ 4. クライアント（静岡市等） ]`

1. **構成案（企画・画コンテ）の承認**: 撮影前にテキスト・構成イメージの段階で確認。
2. **クリエイティブ（写真/動画＋キャプション案）の承認**: `#PR` 表記、ハッシュタグ、リンクを含む最終稿の確認。
3. **最終決裁（弊社承認 ➔ クライアント承認）**: 全承認が下りるまで、インフルエンサー側では絶対に投稿を行わず「下書き保存」状態を維持。

#### 📅 進行スケジュール目標（バッファ重視）
- **初稿（クリエイティブ案）の提出期限**: 投稿予定日の **14営業日前（約2週間〜2.5週間前）** まで（多重確認フローの時間を確保するため厳守）
- **修正差し戻し後の再提出リードタイム**: 指示後 **3営業日以内**
- **NDA・構成合意スケジュール**:
  - NDA締結完了目標: 8月7日（金）まで
  - 構成・台本合意目標: 8月28日（金）まで
  - 現地撮影・編集: 10月中旬〜11月上旬
  - タイアップ投稿実施: 11月9日（月）〜11月13日（金）の期間中（11月中旬の紙媒体配布と連動）

### 3. 法令順守 ＆ トラブル防止対策（ステマ規制対策）
景品表示法（ステルスマーケティング規制）の厳格な遵守と、炎上・稼働不能リスクへの対策を講じます。

- **「タイアップ投稿ラベル」および「#PR」表記の義務化**
  - Instagramのブランドコンテンツツールによる「タイアップ投稿ラベル（タイアップ相手：イベント公式等）」の適用。
  - キャプションの冒頭（スマートフォンの画面でファーストビューに入る位置）に分かりやすく「#PR」と表記することをアサイン側からインフルエンサーに指導・徹底できるか。
- **キャンセルポリシー ＆ 代替アサイン**
  - アサイン決定後、急病等のトラブルでインフルエンサーが稼働できなくなった場合の「代替キャストの無償手配」または「減額返金」ポリシー。
- **事前スクリーニング体制**
  - 炎上リスク（過去の不適切発言、競合イベント等への出演状況など）に関するアサイン会社側の事前調査体制の確認。

---

### 4. 二次利用権・著作権の取り扱い
投稿されたクリエイティブを公式媒体やWEB広告（予算300,000円枠）で活用するための権利範囲を確認します。

- **公式アカウント・特設サイトでの二次利用**
  - 公式Instagramでのリポスト、リール転載の可否および無償期間。
  - 特設WEBサイト等への掲載、またはGoogle/Yahooディスプレイ広告の配信クリエイティブとしての二次利用の可否と追加費用の有無。
  - 二次利用の保証期間（例：イベント終了の2026年11月末まで、または1年間無償など）。

---

### 5. イベント本番（11/28・29）との連携
ツインメッセ静岡でのリアルタイム発信と現場での事故防止策についてすり合わせます。

- **現地リアルタイム投稿（ストーリーズ等）**
  - 会期当日の現地来場およびその場でのストーリーズ投稿が基本プラン内で可能か。
- **現地での撮影マニュアル・プライバシー配慮**
  - 他の一般来場者の顔の映り込みを防ぐための撮影ガイドラインの伝達。
  - 撮影可能なブース・不可のブースの事前マニュアル化。

---

### 6. レポーティング ＆ KPI（成果測定）
実施したタイアップ効果の可視化と次回へのSLA設定を行います。

- **インサイトデータの提出スケジュール**
  - 投稿から何日後に「リーチ数」「保存数」「インプレッション数」「リンククリック数」のキャプチャ（スクショ）を含むレポートが提出されるか（推奨：投稿完了7日後）。
- **エンゲージメント率等の期待値（目標SLA）**
  - 過去の実績値に基づき、今回のターゲット層に対するエンゲージメント率やリーチ数の目標値のすり合わせ。

---

### 7. 連絡体制・コミュニケーション方法の選定
円滑な進行と緊急時の即時対応のため、連絡体制と使用ツールを合意します。

- **日常の進行連絡ツールの選定**
  - アサイン会社担当者と弊社（SAP）間の日常の進捗確認ツール（Slack、LINE、Chatwork、メール等から選定）。
  - インフルエンサーへの直接連絡はアサイン会社担当者を経由することを徹底。
- **緊急連絡フローの確立**
  - 投稿内容の不具合、誤投稿、炎上リスク検知、体調不良による稼働不可などの緊急事態が発生した場合の即時電話連絡先および連絡ルート。
- **大容量ファイル・素材共有方法**
  - 写真・動画素材や構成案テキスト等のやり取りを行うための共有フォルダ（Google Drive等）の開設と権限管理。

---

### 8. 実運用における共通ルール・順守事項
クリエイティブの著作権トラブルやブランドイメージ毀損を防ぐため、以下の共通ルールをインフルエンサーに指導・徹底します。

- **音楽使用（商用ライセンス）の厳格な制限**
  - Instagramリール等の投稿に音楽を載せる場合、二次利用（広告配信・HP掲載）で著作権侵害や警告が発生するのを防ぐため、**「ビジネスミュージックライブラリ（商用利用可能）の無料音源のみ使用」または「音源なし（オリジナル音）」**を義務化。
- **コメント返信・いいね対応ポリシー**
  - 一般ユーザーからのコメントに対するインフルエンサー自身の返信対応の可否（炎上・トラブル防止のため、返信は一律NGまたは事前確認制とし、ポジティブなコメントへのいいね対応のみを推奨等）。
- **投稿維持（削除・非公開の禁止）期間**
  - イベント本番終了後、投稿（フィード、リール等）を削除またはアーカイブ（非公開）にしない最低保証期間の合意（推奨：イベント終了後1年間維持）。
- **ドラフト（下書き）共有時のファイル命名規則**
  - 確認作業時の先祖返りや誤認識を防ぐための、下書きファイル・写真素材のファイル名共通ルールの設定。
