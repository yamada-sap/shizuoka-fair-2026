# 👥 産業フェアしずおか2026 事務局実務担当体制表（裏体制ボード）

本プロジェクトの円滑な進行および主催者（振興協会）・外部協力施工会社との調整を分担するための「実務担当者リスト（裏体制ボード）」です。
