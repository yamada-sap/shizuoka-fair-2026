## 📋 プロジェクト担当・役割別 抜け漏れ防止チェックリスト

### 1. 総合ディレクター (`global-web-promotor`)
* **主要責任**: プロジェクト全体の進行管理、全担当のタスク調整、広告コピー管理。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] 各担当から統括プロデューサーへの承認要請が「業務運営規程ルール通り（直接提示）」行われているか監視。
  * [ ] 進行スケジュールに遅延の兆候（黄信号）が出た際、即座にタスク平準化を実行する。
  * [ ] 広告用テキスト（5案）が各媒体（Google/Yahoo/Insta）に正しくセットされているか目視確認。

### 2. アカウントプランナー (`web-account-planner`)
* **主要責任**: 広告予算配分、CPA/ROI試算、効果シミュレーション。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] インフルエンサーの2万円オーバーを吸収した後の、プロモーション全体予算（52万円または50万円）の均等配分（各約9.3万円など）の再計算。
  * [ ] 各媒体（GDN, YDA, Insta広告）の目標インプレッション数（合計約333万回）の進捗シミュレーションを毎週算出。

### 3. 協賛営業進行 (`ad-sales-manager`)
* **主要責任**: 新規広告「ジャック」営業、協賛企業との調整、入稿管理。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] 協賛企業からの広告協賛品やロゴデータの入稿締切（11月11日）をカレンダーに登録。
  * [ ] 協賛各社への「公式SNSでのメンション協力」の依頼状送付スケジュールを確保。

### 4. 経理・財務担当 (`finance-accounting-manager`)
* **主要責任**: プロジェクト全体の予算・収支管理、出展者追加備品代金の請求・回収。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] 出展者向けの請求元が「静鉄アド・パートナーズ口座」になっており、主催者の口座と明確に区別（誤入金防止）されているか請求書デザインを監査。
  * [ ] 期日未入金（11月13日振込期日）の出展者に対する「督促プログラム」を11月14日より自動稼働させるための文面とフローを事前確定。

### 5. ビジュアルデザイナー (`visual-designer`)
* **主要責任**: ポスター、リーフレット、IDカード、駐車許可証、報告書等の全ビジュアル設計。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] 各種印刷物（リーフレット、案内パンフなど）の用紙選定で「FSC環境配慮用紙・植物性インキ」の仕様が反映されているか印刷会社と確認。
  * [ ] スマートフォンで操作しやすい「スタンプラリー画面のUI/UX」および「写真回収Googleフォーム」のヘッダー画像デザイン。
  * [ ] 搬入時に必須となる「駐車許可証（車両ダッシュボード掲示用）」のA4デザインデータ制作。

### 6. 行政・広報渉外ディレクター (`public-relations-director`)
* **主要責任**: 官公庁届出（消防、警察、保健所）、プレスリリース発信。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] 静岡市消防署への火気使用届（調理ブース用）および、警察署への臨時駐車場・周辺道路使用許可申請のスケジュール確保。
  * [ ] 静岡市保健所への臨時出店（飲食出展）にかかわる事前調整・届出書の回収体制の構築。
  * [ ] 地元メディア（静岡新聞、テレビ静岡等）向けのプレスリリース配信時期（10月中旬・11月上旬）の策定。

### 7. 会場施工・調達監督 (`venue-logistics-director`)
* **主要責任**: 会場施工（望月商事等）の進行管理、搬入ヤード（15分単位）の物流運行。
* **🚨 抜け漏れ防止ToDo**:
  * [ ] 地場観光ゾーンで使用する「木パネル」が、昨年同スペック（W4,200×D1,800×H900）かつ同価格帯で仕入れ可能か、施工会社との見積もり検証（8月中）。
  * [ ] 前日設営日（11月27日）の「15分荷下ろしヤード運行タイムテーブル（ゾーン別時間差）」の作成。
  * [ ] 特装ブース（高さ2.1m超）の自立構造物の転倒防止ガイドライン作成。

### 8. イベント運営責任者 (`event-operations-manager`)
* **主要責任**: 当日進行、ステージMC手配、現場トラブル対応。
* **🚨 抜け漏れ防止ToDo**:

  * [ ] 会場内で発生したクレーム、急病、迷子等のトラブルに対応する「現場運営マニュアル」の作成。

### 9. イベント企画・全体設計担当 (`event-content-planner`)

### 10. 出展者管理・とりまとめ担当 (`exhibitor-coordination-chief`)
