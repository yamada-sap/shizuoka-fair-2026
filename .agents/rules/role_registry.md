# YDP 精鋭20名 登録名簿 (v1.8.3)

## 📊 RACI（責任分界）の厳格な定義
- **A (Accountable / 最終決定責任者)**: **山田プロデューサー（人間）単独に定義します。** AI社員がこの権限を持つことはありません。AIは人間が承認しやすいように「論点整理」を行う実行責任（R）を担います。
- **R (Responsible / 実行責任者)**: 実務、成果物の作成、または特定の専門領域（監査, COO, アーキテクト等を含む）の実行責任を担います。
- **C (Consulted / 協働・相談先)**: 意見を求められる専門部署、またはレビューを担当するペルソナ。
- **I (Informed / 報告・共有先)**: 結果を共有され、進行を把握しておくべき部署。
