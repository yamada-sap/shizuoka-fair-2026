# スキル：2026年度版書類データ自動置換＆タスク抽出

## 1. 厳格な処理ルール
- **カレンダー整合性**：2026年の本番は「11月28日（土）、29日（日）」。すべての日付と曜日を2026年カレンダーに合わせて再計算すること。
- **事務局情報の完全置換**：旧事務局情報を `data/2026_config.txt` に指定された「静鉄アド・パートナーズ（S.A.P.）」の情報に100%置換すること。

## 2. 【絶対厳守】人間用目視確認タスクリストの自動生成
成果物を出力する際、必ず最下部に「🚨 人間での最終確認・作業タスクリスト（手戻り撲滅用）」というMarkdownのチェックリスト（- [ ]）を自動生成すること。
- **カレンダー・曜日チェック**：変換した日付が現実のカレンダーやマスタースケジュールと矛盾していないか確認する指示。
- **ロゴ・画像チェック**：旧ロゴから新ロゴへの差し替え指示。
- **インフラ・数値チェック**：電気容量、消火器ルール、金額などの数値変更に対する専門家（協業会社等）への確認指示。
